# Flappy Bird Genetic Algorithm
- Game Flappy Bird được điều kiển bằng `thuật toán di truyền` để huấn luyện chim bay qua ống.

## Cài đặt
- Cài đặt thư viện `numpy` và `pygame`.
- Chạy lệnh trong terminal `pip install numpy pygame`.

## Thuật toán
- Thuật toán di truyền và cấu trúc mạng `nơ-ron` được cài đặt trong folder `ai`.

## Cách chạy
- Run file `main.py`

## Chế độ
- Người chơi: Nhấn nút `START` để bắt đầu, nhấn phím `Space` để chim nhảy qua cột.
- AI: Nhấn nút `AI` để vào chế độ AI. AI tự động điều khiển chim nhảy qua cột.
